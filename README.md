

AMEDET
=======================


Requirements
------------

In order to develop on this project you need to have the following dependencies

* node 0.10.x
* gulp-cli installed

How to run it
-------------

First, be sure to run ``` npm install ``` to get every dependency needed

* **livereload:** ``` gulp start ```
* **create a ditribution:** ``` gulp dist ```
